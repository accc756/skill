# copy

一个用于生成朋友圈宣传文案的 Codex skill。

适合这些场景：

- AI 平台宣传
- 算力租赁宣传
- AI token 相关业务发声
- 创业者日常朋友圈活跃
- 配海报使用的短文案

这个 skill 的核心风格是：

- 专业发声
- 克制表达
- 不像硬广
- 不说教
- 短句为主
- 尽量避免重复句式

## 适用人群

- 超级个体
- 一人公司
- 初创团队
- 做 AI、算力、数字化服务的人

## 仓库内容

```text
copy/
  SKILL.md
  agents/
    openai.yaml
```

## 安装方式

### 方式一：直接下载仓库

1. 下载这个仓库
2. 找到里面的 `copy` 文件夹
3. 将 `copy` 文件夹放到你的 Codex skills 目录中

常见本地目录示例：

```text
%USERPROFILE%\.codex\skills\
```

放好后，确保最终目录类似这样：

```text
%USERPROFILE%\.codex\skills\copy\
```

### 方式二：克隆仓库

```bash
git clone https://github.com/accc756/skill.git
```

然后把其中的 `copy` 文件夹复制到你的 Codex skills 目录。

## 使用方法

在 Codex 里直接调用：

```text
Use $copy 写一条关于算力租赁的朋友圈文案，专业发声，不要说教。
```

## 调用示例

### 1. 基础版

```text
Use $copy 写10条算力租赁的朋友圈短文案。
```

### 2. 更偏行业观察

```text
Use $copy 写5条偏行业观察的朋友圈文案，主题是 AI 平台和算力服务。
```

### 3. 配海报使用

```text
Use $copy 写8条适合海报搭配的短文案，句子简短，偏专业表达。
```

## 输出特点

默认会尽量做到：

- 读起来像真实从业者
- 有业务相关性
- 适合朋友圈发布
- 不夸张
- 不承诺结果
- 不强行成交
- 适合反复日常使用

## 注意

- 这个 skill 更适合“日常发声”和“持续曝光”
- 如果你需要强销售型文案，建议单独加提示词说明
- 如果你有具体业务、客户对象、海报主题，也可以在调用时直接补充
